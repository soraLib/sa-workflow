# Sa Workflow

A workflow component based on Vue3.

⚠️ `sa-workflow` is currently working in progress. ⚠️

## Features

- [ ] History record
- [ ] Support for horizontal and vertical switching
- [ ] Highly configurable node properties
- [ ] Customizable graph zooming

## Use

```sh
pnpm install
pnpm run dev
```
